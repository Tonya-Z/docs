# 最初のアプリケーションを作成して統合する

まず、[はじめに](./get-started)タブに移動し、右側にある「作成」ボタンを見つけます。このボタンをクリックすると、アプリケーションタブにリダイレクトされます。そこから、「アプリケーションの作成」ボタンをクリックして続行します。

## アプリケーションタイプを選択する

開いたモーダルで、統合したいアプリケーションのタイプを選択します。

<ApplicationTypesIntro />

![Create Application modal](./assets/create-application-modal.png)

利用可能なアプリケーションタイプに対して異なるチュートリアルを用意しており、適切なタイプを選択すると役立ちます。

![Create Application guide](./assets/create-application-guide.png)

## アプリケーション名を入力する

アプリケーション名を入力し、「アプリケーションの作成」をクリックします。例：「書店」。

## Logtoを統合する

タダ！Logtoで最初のアプリケーションを作成しました。詳細な統合ガイドが含まれた祝辞のページが表示されます。ガイドに従って、アプリケーションでのエクスペリエンスを確認します。

:::note
誤って管理コンソールのガイドをスキップした場合、以下の一覧からガイドを選択するか、アプリケーション詳細ページで「ヘルプガイドを確認」をクリックすることもできます。
:::
