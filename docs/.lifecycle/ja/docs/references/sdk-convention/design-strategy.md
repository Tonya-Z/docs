# デザイン戦略

- すべてのプログラミング言語は、`${language}` という単独のgitリポジトリを持つべきです。
- 各プログラミング言語リポジトリはモノリポであるべきです。
- SDKとそれに関連するサンプルプロジェクトは、このリポジトリの下に配置されるべきです。

例:

<details>
  <summary>JS</summary>

- js (core)
- react
- react-sample

</details>

<details>
  <summary>Kotlin</summary>

- kotlin (core)
- android
- android-sample

</details>
