- _Local storage_, also known as [_ConnectorMetadata_](../README.mdx#connectors-remote-storage-connector-db), is an object containing fixed properties such as logo, connector type, and so on. (:face_with_monocle: Having trouble understanding these properties? No worry, a detailed explanation comes later!)
- _Remote storage_ is stored in DB for the sake of relatively frequent changes on those data. 

- _ローカルストレージ_は、_ConnectorMetadata_（../README.mdx#connectors-remote-storage-connector-db）としても知られており、ロゴ、コネクタタイプなどの固定プロパティを含むオブジェクトです。 (:face_with_monocle: これらのプロパティが理解しづらいですか？心配しないで、詳しい説明が後で出てきます！)
- _リモートストレージ_は、それらのデータが比較的頻繁に変更されるためにDBに保存されています。
